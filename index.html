Навчання

Навчаюся на 2 курсі спеціальності "Інженерія програмного забезпечення" у Карпатському національному університеті імені Василя Стефаника. Під час навчання вивчаю основи програмування, алгоритми, структури даних та сучасні підходи до розробки програмного забезпечення.

Програмування

Захоплююся програмуванням та створенням власних програм. Працюю з мовами програмування Python, C++, JavaScript, а також використовую HTML та CSS для створення сучасних веб-інтерфейсів. Постійно вдосконалюю свої навички, створюючи власні проєкти та експериментуючи з новими технологіями.

Проєкти

Розробляю власні програмні проєкти для розвитку практичних навичок.
• Smart Football Planner — програма для управління футбольною командою та планування складу.
• Calculator Pro — десктопний застосунок, створений на C++ та Qt.

Інтереси

Окрім програмування, цікавлюся спортом та сучасними технологіями. Мені цікаво поєднувати програмування зі спортивною тематикою, створюючи корисні та зручні програмні рішення.
